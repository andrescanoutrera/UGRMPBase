{\footnotesize\ttfamily int main (\begin{DoxyParamCaption}\item[{int}]{argc,  }\item[{char $\ast$}]{argv\mbox{[}$\,$\mbox{]} }\end{DoxyParamCaption})}

This program learns a \mbox{\hyperlink{classProfile}{Profile}} model from a set of input DNA files (file1.\+dna, file2.\+dna, ...). The learned \mbox{\hyperlink{classProfile}{Profile}} object is then ordered by frequency and saved in the file output\+Filename (or output.\+prf if the output file is not provided). Running example\+: \begin{quote}
LEARN \mbox{[}-\/t$\vert$-\/b\mbox{]} \mbox{[}-\/k k\+Value\mbox{]} \mbox{[}-\/i profile\+Id\mbox{]} \mbox{[}-\/o output\+Filename\mbox{]} \texorpdfstring{$<$}{<}file1.\+dna\texorpdfstring{$>$}{>} \mbox{[}\texorpdfstring{$<$}{<}file2.\+dna\texorpdfstring{$>$}{>} \texorpdfstring{$<$}{<}file3.\+dna\texorpdfstring{$>$}{>} ....\mbox{]} \end{quote}
